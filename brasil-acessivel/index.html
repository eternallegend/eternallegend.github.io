Brasil Acessível

Plataforma gratuita criada para garantir acesso à cultura, entretenimento e informação para pessoas cegas e com deficiência visual, por meio de conteúdos com audiodescrição e navegação acessível.

Plataformas disponíveis

Web

Acesse diretamente pelo navegador.

Acessar versão web

Windows

Instalador MSI (x64)

Recomendado para a maioria dos usuários do Windows. Instala o Brasil Acessível como um aplicativo tradicional, integrando-se ao menu Iniciar, área de trabalho e lista de programas. Ideal para quem deseja uma instalação simples, automática e segura.

Baixar instalador
Versão portátil

Versão que não requer instalação. Pode ser executada diretamente a partir de um pendrive, HD externo ou pasta local. Indicada para computadores com restrições de instalação ou para uso temporário.

Baixar versão portátil
Instalação via Winget

Método oficial de instalação via gerenciador de pacotes do Windows. Ideal para usuários avançados, técnicos e administradores de sistemas. Permite instalação e atualização automáticas via linha de comando.

winget install azurejoga.brasilacessivel
Instalação via Chocolatey

Método avançado de instalação via Chocolatey. Recomendado para usuários que já utilizam este gerenciador para manter softwares sempre atualizados automaticamente.

choco install brasil-acessivel --version=1.0.0

Linux

AppImage

Formato portátil para Linux. Funciona em praticamente qualquer distribuição sem necessidade de instalação. Basta baixar, tornar executável e abrir.

Baixar AppImage
Debian / Ubuntu (.deb)

Pacote oficial para distribuições baseadas em Debian, como Ubuntu, Linux Mint e derivados. Integra-se ao sistema e ao menu de aplicativos.

Baixar .deb
RPM (Fedora, RHEL, AlmaLinux, Rocky, openSUSE)

Pacote RPM compatível com distribuições baseadas em Red Hat. Permite instalação via gerenciador de pacotes do sistema, garantindo melhor integração e atualizações futuras.

Baixar .rpm
Binário Linux

Executável direto para Linux. Não requer instalação nem permissões administrativas. Ideal para ambientes restritos ou uso avançado.

Baixar binário

Android

Baixar na Play Store

iPhone (iOS – PWA)

No iPhone, o Brasil Acessível funciona como um PWA (Progressive Web App), permitindo uso como aplicativo, mesmo fora da App Store.

  1. Acesse o site no navegador do iPhone
  2. Toque em Compartilhar
  3. Selecione Adicionar à Tela de Início
  4. O app será instalado como um aplicativo normal
