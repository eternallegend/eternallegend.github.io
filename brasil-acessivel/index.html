Brasil Acessível

Plataforma gratuita criada para garantir acesso à cultura, entretenimento e informação para pessoas cegas e com deficiência visual, por meio de conteúdos com audiodescrição e navegação acessível.

Plataformas disponíveis

Windows

Instalador MSI (x64)

Versão recomendada para a maioria dos usuários. Instala o Brasil Acessível no sistema de forma tradicional.

Baixar instalador MSI
Versão portátil (sem instalação)

Ideal para uso em pendrive ou computadores sem permissão de instalação.

Baixar versão portátil
Winget

Comando: winget install azurejoga.brasilacessivel

Chocolatey

Comando: choco install brasil-acessivel --version=1.0.0

Linux

AppImage (todas as distribuições)

Funciona na maioria das distribuições Linux sem instalação.

Baixar AppImage
Debian / Ubuntu (.deb)

Compatível com Debian, Ubuntu e derivados.

Baixar .deb
RPM (Fedora, RHEL, CentOS, AlmaLinux, Rocky, openSUSE)

Pacote RPM compatível com distribuições baseadas em Red Hat e openSUSE (via zypper).

Baixar .rpm
Binário Linux

Execução direta sem gerenciador de pacotes.

Baixar binário

Android

Baixar na Play Store

iPhone (iOS)

  1. Acesse o site no Safari
  2. Toque em Compartilhar
  3. Selecione “Adicionar à Tela de Início”

macOS

Versão em desenvolvimento.
